Getting Started with Stage-Gate

Welcome to the CSIR Stage-Gate Platform

This platform is designed to help you manage technology development projects through the Stage-Gate process, ensuring governance, traceability, and alignment with CSIR standards.

Key Features

  • Project Lifecycle Management: From Stage 0 (Concept) to Stage 3 (Technology Maturation)
  • Gate Reviews: Make Go/Hold/Recycle/Stop decisions with structured scoring criteria
  • Document Management: Upload and track business cases, TRL assessments, IP disclosures, etc.
  • Red Flag System: Automated alerts for delays, budget overruns, and missing resources
  • Reporting Dashboards: Track KPIs, project status, and gate outcomes

User Roles

The system supports different user roles with appropriate permissions:

  • Gatekeepers: Approve/reject projects at each gate
  • Project Leaders: Create and manage projects
  • Team Members: Contribute to project work
  • Process Managers: Configure system templates and workflows

Your First Steps

  1. Complete your profile information in Settings
  2. Explore your Dashboard for active projects and pending reviews
  3. Create a new project or ask to be added to an existing one
  4. Upload required documents as you progress through each stage

Need more help? Visit our FAQs section or contact support.

Managing Projects

Creating a New Project

  1. Navigate to the Projects page
  2. Click the "New Project" button
  3. Fill in the required project information:
    • Project name and description
    • Cluster and research group
    • Initial budget and timeline estimates
  4. Select team members to invite to the project
  5. Submit the form to create your project

Project Lifecycle Stages

The Stage-Gate process consists of four stages:

  • Stage 0 - Concept: Initial idea validation and business case
  • Stage 1 - Research Planning: Detailed plan and feasibility
  • Stage 2 - Feasibility & Development: Technology development
  • Stage 3 - Maturation: Scaling and commercialization readiness

Each stage concludes with a gate review before progressing to the next stage.

Managing Project Details

Once a project is created, you can:

  • Edit basic project information
  • Upload required documents for each stage
  • Monitor progress and metrics
  • Add or remove team members
  • View and respond to red flags

Note: Only Project Leaders and Cluster Managers can create new projects. If you need project creation access, contact your administrator.

Conducting Gate Reviews

The Gate Review Process

At the end of each stage, projects must pass through a gate review before proceeding to the next stage. The review evaluates the project's progress and determines if it meets the criteria to continue.

Gate Review Steps

  1. Submission: Project team submits all required deliverables for the current stage
  2. Scheduling: Gate review meeting is scheduled with appropriate reviewers
  3. Evaluation: Reviewers assess materials and score against predefined criteria
  4. Decision: Gatekeepers make one of four decisions:
    • GO: Project can proceed to next stage
    • HOLD: Temporary pause pending more information
    • RECYCLE: Return to previous stage with revisions required
    • STOP: Terminate project

Scoring Criteria

Each gate review evaluates projects on these key dimensions (weighting may vary by stage):

  • Strategic Alignment (20%)
  • Technical Feasibility (25%)
  • Market Potential (20%)
  • Resource Requirements (15%)
  • IP Position (10%)
  • Risk Assessment (10%)

Scores are aggregated and color-coded for quick assessment (Red/Yellow/Green).

Tip: Review the scoring criteria and weightings before your gate review to ensure you address all key areas in your submission.

Working with Documents

Document Management

The platform allows you to upload, organize, and track documents required for each project stage. Document templates are provided to ensure consistency across projects.

Uploading Documents

  1. Navigate to your project's Documents tab
  2. Click "Upload Document"
  3. Select the document type from the dropdown menu
  4. Drag and drop files or click to browse
  5. Add any comments or notes about the document
  6. Click "Upload" to complete the process

Document Types

Each stage has required and optional documents:

  • Stage 0: Concept brief, Initial business case
  • Stage 1: Research plan, Detailed business case, IP assessment
  • Stage 2: Technical specifications, Prototype results, Market analysis
  • Stage 3: Scaling plan, Commercialization roadmap, Final IP report

Document Status

Documents can have different statuses:

  • Draft: Work in progress, not yet submitted
  • Submitted: Ready for review
  • Approved: Accepted by reviewers
  • Revisions Needed: Requires changes before approval

All documents are version controlled. You can access previous versions from the document history.

Reporting & Analytics

Portfolio Dashboard

The dashboard provides an overview of all projects in the system, including:

  • Project distribution by stage
  • Gate decision statistics
  • Budget utilization
  • Timeline progress
  • Active red flags

Custom Reports

Generate detailed reports on:

  • Project Reports: Detailed status, progress, and financials for individual projects
  • Portfolio Reports: Summary across all projects by cluster, research group, etc.
  • Gate Performance: Analysis of review outcomes and scoring patterns
  • Resource Allocation: Staffing and budget distribution

Reports can be exported to PDF, Excel, or PowerPoint formats.

Key Performance Indicators

The system tracks these metrics automatically:

  • Approval Rate: Percentage of projects receiving GO decisions
  • Stage Duration: Average time spent in each stage
  • Budget Variance: Actual vs. planned expenditures
  • TRL Progression: Technology readiness level advancement

Tip: Use the "Export" function on any dashboard to create snapshots for presentations or external reporting.

Frequently Asked Questions

General Questions

Who can create new projects?

Project Leaders and Cluster Managers can create new projects. If you need project creation access, contact your administrator.

How do I get added to a project?

Project Leaders can add team members. If you need access to a specific project, contact the Project Leader or your manager.

Document Management

What file types can I upload?

You can upload common document types including PDF, Word, Excel, PowerPoint. Size limit is 25MB per file.

Can I edit documents after uploading?

Yes, you can upload new versions of documents. The system maintains version history automatically.

Gate Reviews

Who participates in gate reviews?

Gate reviews typically involve Gatekeepers (decision makers), Project Team (presenters), and often stakeholders from BD&C and IAM teams.

How are gate decisions made?

Decisions are based on scoring against predefined criteria for each stage. The decision (GO/HOLD/RECYCLE/STOP) requires consensus from the Gatekeepers.

Technical Issues

I can't access the system - what should I do?

First, try clearing your browser cache. If that doesn't work, contact the help desk at it-help@csir.co.za or call extension 1234.

How do I report a bug?

Use the "Feedback" button in the bottom right corner of any page to report issues.
